# 🤖 AI Chatbot (College Project)

A smart AI chatbot built using NLP and Machine Learning that understands user queries and provides intelligent responses.

---

## 🧠 Features
- NLP-based chatbot (TF-IDF + ML model)
- Modern UI with animations
- Dark / Light mode
- Voice input 🎤
- Chat memory 💾

---

## 🛠️ Tech Stack
- Python 🐍
- Flask 🌐
- Scikit-learn 🤖
- HTML, CSS, JavaScript 🎨

---

## 📂 Project Structure
ai-chatbot/
│
├── app.py
├── chatbot.py
├── train.py
├── intents.json
├── templates/
│ └── index.html
└── requirements.txt


## How to Run
Install Dependencies ::: pip install -r requirements.txt
Train the Model ::: python train.py
Run ::: python app.py
